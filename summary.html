Parameter Curves¶

2-D Parameter Curves¶

A 2-D Parameter Curve is defined as $f(t) = x, g(t) = y, t [a, b]$

A resulting image set is called a trace of a parameter curve.

As an example, consider the following parameter curve:

$f(t) = cos(t)$

$g(t) = sin(t)$

$t [0, 2\pi] $

In [29]:

import matplotlib.pyplot as plt

t = np.arange(0, 2*np.pi, 0.001)
x = np.cos(t)
y = np.sin(t)

plt.axis('equal')
plt.grid()
plt.plot(x, y)
Out[29]:
[<matplotlib.lines.Line2D at 0x7fd0b24c10d0>]

The trace of the Parameter Curve is the unit circle. If we change the Parameter Curve to:

$f(t) = cos(2\pi t)$

$g(t) = sin(2\pi t)$

$t [0, 1] $

The trace of the Parameter Curve is the unit circle again. So we see, the trace of a parameter curve is independent of the parameterization.

Velocity and Acceleration¶

The interesting thing now is the speed at which the circle is traversed.

The velocity is defined as:

$K'(t): [a, b] \rightarrow \mathbb{R}^2, t \mapsto \begin{pmatrix} f'(t) \\ g'(t) \end{pmatrix}$

The web speed is defined as:

$v(t) = || K'(t) ||$

The acceleration is defined as:

$K''(t): [a, b] \rightarrow \mathbb{R}^2, t \mapsto \begin{pmatrix} f''(t) \\ g''(t) \end{pmatrix}$

Curves with web speed=1¶

Another special type of curves are those with a constant velocity of 1. These are called unit speed curves. An example is the following:

$f(t) = r * cos(\frac{t}{r})$

$g(t) = r * sin(\frac{t}{r})$

$t [0, \mathbb{2\pi r}]$

Tangent and Normal¶

Now we know about velocity and acceleration, we can define the tangent and normal to a curve.

The tangent or tangent vector is defined as:

$\vec T(t) = \frac{1}{v(t)} \begin{pmatrix} f'(t) \\ g'(t) \end{pmatrix}$

We can also define the normal or normal vector as:

$\vec N(t) = \frac{1}{v(t)} \begin{pmatrix} -g'(t) \\ f'(t) \end{pmatrix}$

The tangent and normal are orthogonal to each other. Note that the normal vector rotates the tangent vector left by $90^{\circ}$. You can use the right hand rule to verify this.

Here's an example of the unit circle with some tangents and normals:
